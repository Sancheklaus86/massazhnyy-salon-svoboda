# Массажный салон Свобода - Project Overview

## 🎯 Цель проекта

Создание одностраничного showcase-сайта для массажного салона "Свобода" в Тольятти, который демонстрирует возможности современной веб-разработки (вайбкодинга) и продвигает услуги массажа.

**Двойная функция:**
1. **Демонстрационный проект** - витрина современных веб-технологий, анимаций и интерактива
2. **Коммерческий сайт** - продающая страница услуг массажного салона

## 👥 Целевая аудитория

**Основная аудитория:**
- Жители Тольятти, интересующиеся услугами массажа
- Возраст: 25-55 лет
- Люди, заботящиеся о здоровье и релаксации
- Активные пользователи интернета

**Вторичная аудитория:**
- Веб-разработчики, изучающие современные технологии
- Потенциальные клиенты для веб-студий
- HR специалисты, оценивающие портфолио

## 🎨 Концепция дизайна

**Философия:** "Каждый блок — отдельная визуальная история"

**Ключевые принципы:**
- **Смелость** - яркие акценты, современные эффекты
- **Минимализм** - много "воздуха", чистые композиции
- **Интерактивность** - сайт "живой", реагирует на действия
- **Современность** - тренды 2025-2026 года

## 🌟 Уникальные фичи

1. **Переключение тем** - тёмная и светлая тема
2. **Scroll-triggered анимации** - элементы оживают при прокрутке
3. **Интерактивные элементы** - кнопки с эффектами, слайдеры, drag-and-drop
4. **Visual showcase** - каждая секция демонстрирует отдельную технологию
5. **Device preview** - демонстрация адаптивности в реальном времени
